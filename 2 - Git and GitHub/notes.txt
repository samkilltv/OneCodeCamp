Check for git version
Command: git --version

Configure Git
Command: git config --global user.name "username"
Command: git config --global user.email "email"

Initialize an empty repository
git init

Check for the status of files
Command: git status

Add files to the staging area
Command: git add .

Create a commit
Command: git commit -m "This is the description."

Show the history
Author, Date, description
Command: git log
