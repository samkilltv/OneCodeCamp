Check for git version
Command: git --version

Configure Git
Command: git config --global user.name "username"
Command: git config --global user.email "email"

Initialize an empty repository
git init

Check for the status of files
Command: git status

Add files to the staging area
Command: git add .

Create a commit
Command: git commit -m "This is the description."

Bypass the staging area
Command: git commit -a -m "This is the description"

Show the history
Author, Date, description
Command: git log

Description 
Command: git log ---oneline

This modification is already done

git remote add origin [repository-url]

List all of the branches
git branch -a

Create a branche
Command: git branch [branch-name]

Switch to a different branch
Command: git switch [branch-name]
